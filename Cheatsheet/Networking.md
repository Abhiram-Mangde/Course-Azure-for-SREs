# Networking - CheatSheet

## Basic Networking Command

1. ip addr - Display or configure IP addresses
2. ip route - Show or manipulate routing tables
3. traceroute - Trace the route packets take to a host
4. nslookup - Query DNS records
5. dig - Query DNS servers
6. ssh - Connect to a remote server via SSH 
7. ftp - Transfer files using the FTP protocol
8. nmap - Network scanning and discovery 
9. telnet - Communicate with remote hosts
10. netcat - Read/Write data over networks
