# Linux - CheatSheet

Linux is the foundation of DevOps operation. These commands help you navigate systems, manage files, configure premissions and automate tasks in terminal environments. 

## Basic Linux Command

1. `pwd` - Print the current working directory
2. `ls` - List files and directories
3. `cd` - Change directory
4. `touch` - Create an empty file
5. `mkdir` - Create a new directory
6. `rm` - Remove files or directories
7. `rmdir` - Remove empty directories
8. `cp` - Copy files or directories
9. `mv` - Move or rename files and directories
10. `cat` - Display the content of a file
11. `echo` - Clear the terminal screen

## Intermediate Linux Command

1. `chmod` - Change file permissions
2. `chown` - Change file ownership
3. `find` - Search for files and directories
4. `grep` - Search for text in file
5. w`c - Count lines, words and characters in a file.
6. `head` - Display the first few lines of a file
7. `tail` - Display the last few lines of a file
8. `sort` - Sort the content of a file
9. `uniq` - Remove duplicate lines from a file 
10. `diff` - Compare two files line by line
11. `tar` - Archive files into a tarball.
12. `zip/unzip` - Compress and extract ZIP files
13. `df` - Display disk space usage
14. `du` - Display directory size
15. `top` - Monitor system processes in real time
16. `ps` - Display active processes
17. `kill` - Terminate a process by its PID
18. `ping` - Check network connectivity
19. `wget` - Download files from the internet
20. `curl` - Transfer data from or to a server
21. `scp` - Securely copy files between systems
22. `rsync` - Synchronize files and directories
