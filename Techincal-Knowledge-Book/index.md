# 1. Microsoft AZ-900 (Microsoft Azure Fundamentals)

AZ-900 is a foundational certification from Microsoft. It is the starting point for a career in Cloud Computing with Microsoft Azure.

**Helps to:**
- Understand how cloud systems work and basic cloud computing concepts.
- Work with Azure environments and understand Microsoft Azure services.
- Understand how Azure is managed, secured, and governed.

**Divided into three najor learning modules:**
1. Introduction to Cloud Infrastructure: Describe cloud concepts
2. Introduction to Cloud Infrastructure: Describe Azure architecture and services
3. Introduction to Cloud Infrastructure: Describe Azure management and governance

**Concepts Learnt:**
1. Cloud Service Models (IaaS, PaaS, SaaS)
2. Cloud Deployment Models (Public, Private, Hybrid)
3. Core Azure Architectural Components (Region, Availability Zone, Resource Group, Subscription)
4. Core Azure Services (Compute, Networking, Storage)
5. Security (Azure AD, RBAC, Defender)
6. Governance Tools (Azure Policy, Resource Locks, Blueprints)
7. Management Tools (Portal, CLI, PowerShell, Monitor)

**After Completion what you should know or are expected to answer:**
- Explain cloud computing.
- Compare IaaS, PaaS, SaaS.
- Design basic Azure architecture.
- Understand how Azure regions and availability zones work.
- Deploy and organize resources using resource groups.
- Understand Azure billing structure.
- Use Azure management tools.
- Apply basic security and governance principles.

> AZ-900 gave me strong foundational knowledge of cloud computing and Microsoft Azure. I learned about cloud service models, deployment models, Azure architecture components, compute, networking, and storage services. I also understand Azure security, governance, and management tools. It built my base for working in cloud environments and preparing for advanced Azure certifications.

# 2. Microsoft AI-900 (Azure AI Fundamentals)

AI-900 is a foundational certification from Microsoft. It builds awareness and understanding of AI solutions in Azure.

**Helps to:**
- Understand AI concepts and Machine Learning fundamentals.
- Know AI services in Azure.
- Identify which Azure service is used for which AI solution.

**Concepts Learnt:**
1. Introduction to Artificial Intelligence (AI)
2. Machine Learning (Supervised, Unsupervised, Reinforcement)
3. Computer Vision
4. Natural Language Processing (NLP)
5. Speech AI
6. Generative AI
7. Azure AI Services (Azure Machine Learning, Azure AI Services, Azure AI Bot Service)

**After Completion what you should know or are expected to answer:**
1. Explain AI Concepts & Differentiate between ML, NLP, Computer Vision, and Generative AI.
2. Understand basic ML model concepts.
3. Understand Responsible AI principles.
4. Understand how AI integrates with cloud systems.
5. Basic understanding of data handling.
6. Awareness of training and testing datasets.
7. Understanding of model evaluation.
8. Understanding of AI pipelines.
9. Awareness of how APIs are used in AI services.
10. Basic understanding of client-server architecture in AI solutions.

> AI-900 gave me foundational knowledge of Artificial Intelligence and Machine Learning concepts. I learned about different AI workloads such as Machine Learning, Computer Vision, NLP, Speech, and Generative AI. I also understand which Azure AI services are used to build these solutions. Additionally, I gained knowledge about Responsible AI principles and how AI integrates with cloud platforms.

# 3.Microsoft DP-900 (Azure Data Fundamentals)

DP-900 is a foundational certification from Microsoft focused on Data concepts in Azure. It builds a strong foundation in cloud data technologies.

**Helps to:**
- Understand core data concepts, how data is stored and managed.
- Understand different types of databases.
- Understand relational and non-relational data.
- Understand Azure data services.
- Understand analytics and big data basics.

**Concepts Learnt:**

1. What is Data
2. Types of Data (Structured, Semi-Structured, Unstructured)
3. Relational Data
4. Azure Relational Database Services (Azure SQL Database, Azure SQL Managed Instance, Azure Database for PostgreSQL / MySQL)
5. When to Use Relational Databases
6. Non-Relational Data (NoSQL)
7. Azure Storage for Non-Relational Data (Azure Blob Storage, Azure Table Storage, Azure Files)
8. Azure Cosmos DB
9. Big Data Concepts (Volume, Velocity, Variety)
10. Data Analytics Concepts (Descriptive Analytics, Diagnostic Analytics, Predictive Analytics, Prescriptive Analytics)
11. Azure Analytics Services (Azure Synapse Analytics, Azure Data Factory, Azure Stream Analytics, Power BI)

**After Completion what you should know or are expected to answer:**
- Explain structured vs unstructured data.
- Database design principles.
- Data storage models.
- Cloud database scalability.
- High availability in data systems.
- Data integration workflows.
- Real-time vs batch processing.
- Analytical workloads vs transactional workloads.

> DP-900 gave me foundational knowledge of data concepts in a cloud environment. I learned about relational and non-relational databases, Azure SQL services, Azure Cosmos DB, big data concepts, data warehousing, ETL processes, and data visualization using Power BI. It helped me understand how data is stored, processed, and analyzed in Azure.
